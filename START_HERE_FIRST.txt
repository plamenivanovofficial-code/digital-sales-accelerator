⚡ БЪРЗ СТАРТ (5 СТЪПКИ):

1️⃣ SETUP .ENV FILE
   cp .env.template .env
   
   Редактирай .env и добави Gemini API key:
   GEMINI_API_KEY=your_api_key_here
   
   Вземи безплатен key от:
   https://makersuite.google.com/app/apikey

2️⃣ INSTALL DEPENDENCIES
   pip install -r requirements.txt

3️⃣ (OPTIONAL) SETUP OLLAMA
   За bulletproof AI:
   
   ollama serve
   ollama pull llama3.1:8b
   
   Ollama е optional, но СИЛНО препоръчителен!

4️⃣ INITIALIZE DATABASE
   python update_database.py
   python test_integration.py

5️⃣ START OMEGA
   streamlit run app.py
   
   или двоен клик на START.bat (Windows)
